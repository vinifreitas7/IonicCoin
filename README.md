IonicCoin

Aplicativo de conversão de moedas desenvolvido com Ionic e Angular.

Objetivo

Permitir a conversão de moedas em tempo real utilizando uma API REST de câmbio.

Funcionalidades

* Conversão de moedas em tempo real
* Integração com ExchangeRate API
* Conversão inversa
* Histórico de conversões
* Funcionamento offline com taxas armazenadas localmente
* Suporte a múltiplas moedas

Tecnologias Utilizadas

* Ionic
* Angular
* TypeScript
* ExchangeRate API
* LocalStorage

Como Executar

1. Instale as dependências:

npm install

2. Execute o projeto:

ionic serve
